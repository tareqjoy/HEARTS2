# HEARTS

An android app of the card game HEARTS. To know more about the hearts game: https://en.wikipedia.org/wiki/Hearts_(card_game)

# FEATURES

> Play with friends (using internet)
> Play with AI Bot
> Great UI
> Smooth animation
> Secure game movement (not hackable mechanism)
